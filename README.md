# Teslo API

1. Clonar proyecto
2. npm install
3. Clonar el archivo .env.example y renombrarlo a .env
4. Cambiar las variables de entonrno

5. Levantar la bd

```
docker-compose up -d
```

6. Levantar proyecto npm run start:dev

7. Ejecutar SEED `http://localhost:3000/api/seed`
